# Orion Collective Orchestration System (OCS)

## Overview
The OCS is a fully adaptive, self-evolving orchestration system designed to transform high-level intent into high-fidelity execution.

## Structure
- `src/api`: FastAPI backend services
- `src/cli`: Command-line tools
- `src/shared`: Shared libraries and models
- `docs/blueprint`: System architecture and planning documents
- `docs/research`: Research notes and educational material

## Development
This project follows modern Python standards.
- Python 3.12+
- PEP 621 Configuration
- Use `uv`, `pdm`, or `pip` to install dependencies.
